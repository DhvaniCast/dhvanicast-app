## Color Usage Guidelines:

### ‚úÖ **DO:**
- Use `0xFF00ff88` (Neon Green) for primary actions
- Use `0xFFff4444` (Red) for destructive actions
- Use dark backgrounds (`0xFF1a1a1a`, `0xFF2a2a2a`) for cards
- Use gray variants for borders and inactive states

### ‚ùå **DON'T:**
- Don't use red for positive actions
- Don't mix too many accent colors in one screen
- Avoid using pure white on dark backgrounds (use grays instead)
